# 人脸识别签到欢迎系统  
## 一.老师要求：  
整体代码量大于1000行，每个人都需要编写一部分代码。  

## 二.系统环境：  
语言C#  
百度人脸识别，语音合成SDK集成  
SDK文档地址https://ai.baidu.com/docs#/Face-Csharp-SDK/top  
团队成员：A、B、C、D、E、F  
开发IDE：Visual Studio  
GitHub地址https://github.com/huisoftware/FaceCheck.git  

## 三.模块和需求：  
一级菜单（签到，人脸库维护）  

### 1.签到界面：  
摄像头实时拍摄-点击拍照按钮-语音提示xxx识别成功  

### 2.人脸库维护界面  
查询用户组列表（id即名字，英文）  
新增用户组  
删除用户组  
使用用户组（最多同时使用10个，就是勾选识别的范围）  
停止使用用户组  
查询用户组下用户  
重置签到状态  
用户组下新增用户(用户人脸注册(第一张替换，其他张追加))  
用户组下删除用户  
用户人脸更新（第一张替换，其他张追加）  


## 四.任务分解：  
A|1.页面  
A|2.摄像头持续拍摄  
A|3.拍照保存图片  
A|3.调用百度接口识别人脸，结果解析  
A|4.语音提示  
D|5.查询用户组列表  
B|6.新增用户组  
B|7.删除用户组  
D|8.使用用户组（最多同时使用10个，就是勾选识别的范围，list中保存）  
D|9.停止使用某用户组（list中删除）  
F|10.查询用户组下用户（已签到的标出来，已签到的id在一个list里）  
C|11.重置签到状态（所有人改为未签到，一个id的list维护，不持久化）  
C|12.用户组下新增用户(用户人脸注册(第一张替换，其他张追加)，选择多张图片，如果不能选择多张，则单张选择后保存在已选列表中，最后一并确认)  
F|13.用户组下删除用户  
C|15.用户人脸更新（第一张替换，其他张追加，选择多张图片，如果不能选择多张，则单张选择后保存在已选列表中，最后一并确认）  
B|16.ppt制作  
B|17.测试
